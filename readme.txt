Lord Linus Online Visitor Plugin show the total number of Online users showing at the moment Besides that you can show the IP address of the users too. 

== Description ==
Lord Linus Online Visitor plusing shows the total number of Online users that are available on your site. Besides that for the help of your users, you can show the IP address of the user on the site there too.

If the admin needs that he want to show the total number of visitors on his website, then he can show it.

This plugin is in an initial stage which will come to you with the following features very soon.

1. Users can see the total Users based on country
2. Admin can chat with the online users there without any account on outer side.
3. Multilingual

The Demo of the plugin is available on the Demo page of http://swm99shop.in

To know more about the plugin you can go to the site http://swm99shop.in or you can send mail at ucerturohit@gmail.com

== Installation ==

Here is what you need to do: 

1. Install plugin OR unzip the downloaded file to /wp-content/plugins/
1. Go to your plugin settings and enable the plugin.
1. In the sidebar you will have a new section called 'Online Visitors' navigate there and set your preferences.
1. Go to Appearance=> Widgets section to put the widget on your site.
1. To read the translation menu, you can go to http://impulsesoftech.in/translatingplugin.pdf
